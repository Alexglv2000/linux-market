# Contribuyendo a Linux-Market POS

¡Gracias por tu interés en colaborar con Linux-Market POS! Este proyecto es de código abierto y agradecemos cualquier tipo de contribución.

## Cómo empezar

1. **Haz un Fork** del repositorio.
2. **Clona tu fork** localmente.
   ```bash
   git clone https://github.com/TU_USUARIO/linux-market.git
   ```
3. **Crea una rama** para tu cambio.
   ```bash
   git checkout -b feature/mi-nueva-funcionalidad
   ```
4. **Realiza tus cambios** y haz commits claros.
5. **Haz Push** a tu rama.
   ```bash
   git push origin feature/mi-nueva-funcionalidad
   ```
6. **Abre un Pull Request** explicando tus cambios.

## Áreas de Contribución

- **Traducciones**: Ayúdanos a traducir el sistema a otros idiomas.
- **Documentación**: Mejora el manual de usuario o la guía técnica.
- **UI/UX**: Sugiere mejoras en el diseño o implementa nuevos componentes.
- **Backend (Rust/Tauri)**: Optimiza el rendimiento o agrega integración nativa.
- **Seguridad**: Reporta vulnerabilidades y ayúdanos a corregirlas.

## Estilo de Código

- Usamos **Next.js** con TypeScript.
- Estilos con **Tailwind CSS**.
- Componentes basados en **Shadcn UI**.
- Por favor, mantén un código limpio y comentado cuando sea necesario.
